You can add Title, Subtitle and Caption to the option's definition as follows: 
$scope.options = {
    // chart options
    chart: {
        type: 'lineChart',
        ...
    },

    // title options
    title: {
        enable: true,
        text: 'Title for Line Chart'
    },

    // subtitle options
    subtitle: {
        enable: true,
        text: 'Subtitle for simple line chart. Lorem ipsum dolor sit amet...',
        css: {
            'text-align': 'center',
            'margin': '10px 13px 0px 7px'
        }
    },

    // caption options
    caption: {
        enable: true,
        html: 'Figure 1. Lorem ipsum dolor sit amet...',
        css: {
            'text-align': 'justify',
            'margin': '10px 13px 0px 7px'
        }
    }
}
Title, Subtitle and Caption can contain the following options:
Option Description
enable true: boolean. Use / don't use.
text Some text.: string. Used as a plain text.
html Some html.: string. Used as a html instead of a plain text.
className h4: string. Add any class definition.
css {}: object. Add additional styles for html.
