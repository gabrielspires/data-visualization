Zooming functionality [since v1.0.3+] is available in the following charts:
  • Candlestick Bar Chart
  • Cumulative Line Chart
  • Historical Bar Chart
  • Line Chart
  • OHLC Bar Chart
  • Scatter Chart
  • Scatter Plus Line Chart
  • Stacked Area Chart
chart: {
    /* Chart options ... */

    zoom: {
        //NOTE: All attributes below are optional
        enabled: true,
        scale: 1,
        scaleExtent: [1, 10],
        translate: [0, 0],
        useFixedDomain: false,
        useNiceScale: false,
        horizontalOff: false,
        verticalOff: false,
        zoomed: function(xDomain, yDomain) {
            var domains = {x1: 0, x2: 0, y1: 1, y2: 1};
            return domains;
        },
        unzoomed: function(xDomain, yDomain) {
            var domains = {x1: 0, x2: 0, y1: 0, y2: 0};
            return domains;
        },
        unzoomEventType: 'dblclick.zoom'
    }
}
Option Description
enabled true: boolean. Turn on / off zooming.
scale 1: number. Initial scale factor.
scaleExtent [1, 10]: array.
Specifies the zoom scale's allowed range as a two-element array, [minimum, maximum].
translate [0, 0]: array. Specifies the current zoom translation vector.
useFixedDomain false: boolean. Restrict zooming area to only available data range.
useNiceScale false: boolean. Extend the scale domain to nice round numbers.
horizontalOff false: boolean. Turn off horizontal zoom.
verticalOff false: boolean. Turn off vertical zoom.
zoomed(xDomain, yDomain) noop: function. Zoom event handler.
Input: xDomain: [lowX, highX], yDomain: [lowY, highY].
Output: {x1: lowX, x2: highX, y1: lowY, y2: highY}.
unzoomed(xDomain, yDomain) noop: function. Unzoom event handler.
Input: xDomain: [lowX, highX], yDomain: [lowY, highY].
Output: {x1: lowX, x2: highX, y1: lowY, y2: highY}.
unzoomEventType "dblclick.zoom": string. Which event type use to reset zoom to initial state. Available events:
mousedown.zoom
mousewheel.zoom
mousemove.zoom
DOMMouseScroll.zoom
dblclick.zoom
touchstart.zoom
touchmove.zoom
touchend.zoom
none
Note that all options are optional, so to include zooming is enough to define an empty zoom options: 
chart: {
    /* Chart options ... */

    zoom: {}
}
